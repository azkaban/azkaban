Building from Source

Azkaban builds use Gradle, and requires a Java JDK, version 8 or higher. Actually, users don't need to install gradle themselves, since we have a gradle wrapper in our repository.

on Unix-like platforms such as Linux and Mac OS X

$ ./gradelw distTar

on Windows using the gradlew.bat batch file

$ gradlew distTar
Gradle creates .tar.gz files inside project directories. eg. ./azkaban-solo-server/build/distributions/azkaban-solo-server-0.1.0-SNAPSHOT.tar.gz. You can now un-archive the distribution using tar xvzf azkaban....tar.gz.
